### 2. Frontend — session behavior (LinguaFormula parity)

- **`sessionStorage` session flag** (`davidlong_tech_auth_session`): On initial load, if the flag is missing (e.g. new tab after closing the last one), the app clears the JWT, **`POST /api/auth/logout`** to clear the httpOnly cookie, and skips `GET /api/auth/me` so the user is not restored from cookie alone. Flag is set on successful login, register, and successful `/me`; cleared on logout.
